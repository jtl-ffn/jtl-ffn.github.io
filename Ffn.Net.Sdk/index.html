FFN SDK Documentation

This is the official C# SDK for connecting to the Fulfillment Network.

REST API

The Fulfillment Network provides a REST API to connect merchants and fulfillers with each other.
The purpose of this SDK is to provide a simple set of APIs to create applications that connect to the network either as a merchant or as a fulfiller.
